# 🦖 Dino AI Controller (Hand Gesture Game)

Game điều khiển nhân vật bằng **cử chỉ tay (AI + Computer Vision)** sử dụng MediaPipe và OpenCV.

## 🚀 Tính năng
- Điều khiển nhân vật bằng tay qua webcam
- Nhận diện cử chỉ:
  - ✊ ROCK → nhảy
  - 🖐 PAPER → reset game khi thua
- Game chạy bằng Pygame
- Xóa background người bằng AI (MediaPipe Selfie Segmentation)
- Hiển thị camera realtime

---

## 🧠 Công nghệ sử dụng
- Python 3.11
- OpenCV
- MediaPipe
- Pygame
- NumPy

---
🎮 Cách chơi

Giơ ✊ ROCK (nắm tay) → nhân vật nhảy

Giơ 🖐 PAPER (xòe tay) → reset khi thua

Tránh chướng ngại vật

Nhấn SPACE để restart
